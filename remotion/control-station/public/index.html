    Rendering speed

    This machine has an NVIDIA GPU (RTX 4070). NVENC gives a confirmed ~4.7x faster web-delivery encode, verified by timing both. Note: "Hardware acceleration" below currently has no effect on the master render itself. Remotion disables it whenever a CRF quality target is set, which we always do. The real lever for render speed is Concurrency, since that parallelizes frame capture, the actual bottleneck. NVENC's file sizes also run larger than libx264 at the same nominal quality number: a real trade-off, not a bug.
